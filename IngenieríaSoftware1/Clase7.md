# Términos utilizados en clase
## Sistema SCADA
Un sistema SCADA (Supervisory Control And Data Acquisition) es un sistema que se usa para supervisar, controlar y recopilar datos de procesos industriales en tiempo real.

>Es como un  "centro de control" que permite ver y manejar máquinas o procesos a distancia.

## Función
+ Monitorear variables (temperatura, presión, nivel, etc.)
+ Controlar equipos (válvulas, motores, bombas)
+ Detectar fallos o alertas
+ Guardar datos históricos

## ¿Dónde se usa?
+ Plantas eléctricas 
+ Sistemas de agua 
+ Fábricas 
+ Petróleo y gas

## Componentes básicos
+ Sensores → recogen datos
+ PLC/RTU → procesan datos
+ Software SCADA → muestra todo en pantalla
+ HMI → interfaz para el usuario
